# Proyecto Taxímetro

Primer proyecto desarrollado en Python para el bootcamp de Factoría F5.

## Objetivo

Desarrollar un sistema básico de taxímetro capaz de calcular tarifas según:

- Tiempo detenido
- Tiempo en movimiento

## Tecnologías utilizadas

- Python
- Git
- GitHub
- VS Code
